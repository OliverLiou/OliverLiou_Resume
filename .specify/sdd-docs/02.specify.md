# 建立個人履歷網站需求
- Header 區塊需要包含網站標題及導航選單
- Footer 區塊需要包含版權資訊及社群連結(mail、github、linkedin)
- body 區塊, 請根據以下順序及內容建立
    - 自我介紹: 圖片、姓名、職稱、簡短介紹
    - 技能清單: 前端技能、後端技能、其他技能
    - 工作經歷: 公司名稱、職稱、工作時間、公司logo
    - 專案經驗: 專案名稱、專案描述、使用技術、開發區間(始和末)
    - 教育背景: 學校名稱、學位、就讀時間
- 必須能夠在手機及桌面裝置上有良好的顯示
- 需要以現在商務白簡潔風格設計, 並且支援深色模式
- 所有內容都需要支援中英文切換
