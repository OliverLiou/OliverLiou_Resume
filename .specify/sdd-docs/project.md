# 專案經驗

- 快速報價系統 (Quickly Price Quotation Program)
    - 專案描述: 根據Bom上的物料清單, 自動至ERP系統蒐集相關成本, 並根據公式計算成本, 最終生成公司的標準成本分析表。
    - 開始時間: 2024-12
    - 結束時間: 2025-03
    - 使用技術: .Net 8、Nuxt 3
    - 功能描述:
      - Bom物料清單上傳
      - 跨系統與Oracle EBS系統進行資料整合
      - 根據預設公式計算成本
      - 生成標準成本分析表

- 鋼捲管理系統 (Steel Coil Management System)
    - 專案描述: 與中國鋼鐵公司進行業務整合，處理完整的鋼材貿易流程
    - 開始時間: 2022-08
    - 結束時間: 2023-10
    - 使用技術: .Net 6、Nuxt 2
    - 功能描述:
        - 紀錄鋼材採購、合約、發票、押匯資料
        - 定時任務, 自動更新與追蹤資料
        - 根據物料需求產生推移表, 並預測下次採購時間

- 節拍生產排程系統 (Tempo Production Auto Schedule System)
    - 專案描述: 以生產線別為單位, 根據每月的預示量, 進行生產排程的管理與控制, 避免庫存過多或缺料, 提升製造效率並實現生產排程控制
    - 開始時間: 2021-09
    - 結束時間: 2022-07
    - 使用技術: .Net 5、Nuxt 2
    - 功能描述:
        - 行事曆管理
        - 支援多種排程策略(特急、連續、區間、定頻…等等)
        - 各式報表輸出(月排程、達成率…等等)
        - 跨系統與Oracle EBS系統進行即時生產監控

- 公司前端/後端開發模板(Frontend/Backend Development Template)
    - 專案描述: 公司內部前端與後端開發模板, 縮短專案開發時間及提升開發效率
    - 開始時間: 2024-03
    - 結束時間: 持續更新中
    - 使用技術: .Net 8 、Nuxt 3
    - 功能描述: 
        - 權限控制
        - Entity Log日誌記錄
        - 與內部AD整合實現單一入口登入
        - 統一資料表CRUD操作
        - 統一前端 layout
        - 頁籤導航
